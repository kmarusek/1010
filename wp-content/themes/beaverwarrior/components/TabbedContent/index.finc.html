TabbedContent component

Example tabbed content

TabbedContent is useful for allowing the user to select multiple content bits. It should not be used to hide important information that most users would be interested in just for the sake of fitting things above the scroll fold.

This particular specimen has a selector list with the --centered qualifier, which spreads the tabs around the width of the list.
