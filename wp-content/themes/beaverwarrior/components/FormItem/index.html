FormItem component

Ordinary form elements

Suitable for long lists of full-width form items. Use the FormItem--actions variant to markup a string of submit/cancel buttons.

Layout caution

        
When applying custom layouts to an --actions block, remember to set the @Grid--gutter_width of any .make-Column--gutter declarations equal to that of @FormItem--action--spacing_anomaly or the buttons within the --actions block will not align with other form fields.
