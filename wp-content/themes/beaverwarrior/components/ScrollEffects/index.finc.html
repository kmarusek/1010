ScrollEffects component

CSS-based scroll effects

These effects are triggered by special state classes provided by the ScrollEffects script.

ScrollEffects--appear

The --appear effect causes content to transition from opacity 0 to opacity 1 when visible. The visible effect mode allows the content to disappear once more when it is scrolled off the page.
