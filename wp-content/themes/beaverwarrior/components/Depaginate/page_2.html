Depaginate component

Example depaginate instance

Depaginate allows you to automatically load additional pages of a multiple page view generated by a dynamic backend onto the same page. Features include:

  • Extremely simple setup (requires only data attributes on an existing pager)
  • No client-side rendering required
  • Preserves ability to link to specific pages
  • Preserves user history upon clicking back button
  • Supports scroll-back for loading previous pages
